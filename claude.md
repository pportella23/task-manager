# Task Manager — Week 1

See `ai-context/frontend-mentor.md` for full mentorship context, behavioral rules, and study system.

## This project
Week 1 of a 12-week React mastery plan targeting a SAP Concur Frontend Developer role.

## Hard constraints for this project
- No libraries beyond what is in package.json (React, TypeScript, Vite)
- No routing
- All task state in App.tsx only — no external state management
- Styled with plain CSS only
- TypeScript strict mode — zero `any`, zero `ts-ignore`
- Filtered tasks are always derived, never stored in state

## Commands
- `npm run dev` — start dev server
- `npm run lint` — run ESLint (must pass before any commit)
- `npm run format` — run Prettier
- `tsc --noEmit` — typecheck without building
